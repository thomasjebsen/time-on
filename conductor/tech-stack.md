# Tech Stack

## Language

- **Swift** (Swift Package Manager)

## Framework

- **AppKit** — native macOS menu bar app
- Open to SwiftUI where it fits; no strong framework preference

## Database

- **UserDefaults** for preferences
- No strong opinion on persistence for history — use whatever fits the need

## Distribution

- **Homebrew Cask** + **GitHub Releases**
- Built and released via GitHub Actions

## Key Dependencies

- macOS native APIs (IOKit for caffeinate/keep-awake, AppKit for UI)
