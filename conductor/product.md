# Product Definition

## Project Name

Time On

## Description

A macOS menu bar app that helps you stay aware of your active session time and keeps the computer awake if needed — combining the best of [Pandan](https://sindresorhus.com/pandan) (screen time tracking) and [Caffeine](https://intelliscapesolutions.com/apps/caffeine) (keep-awake) in one tool.

## Problem Statement

It's easy to lose track of how long you've been working, leading to poor time awareness and burnout. Users also need to prevent their Mac from sleeping during long-running jobs and presentations. Existing tools solve these separately — Time On combines both.

## Target Users

Anyone who needs to track active work sessions and prevent Mac sleep.

## Key Goals

1. **Simple, unobtrusive time awareness** — display session time in the menu bar without getting in the way
2. **Reliable keep-awake functionality** — prevent Mac sleep for presentations and long tasks
3. **Nice extras** — emoji customization, session history, and other quality-of-life features
4. **Privacy first** — all data stays on the machine. No accounts, no telemetry, no internet access
